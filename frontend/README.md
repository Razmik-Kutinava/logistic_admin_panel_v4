# Frontend - Delivery App

## Описание

Frontend часть приложения доставки.

## Установка

```bash
npm install
```

## Настройка

1. Скопируйте `.env.example` в `.env`
2. Заполните необходимые переменные окружения

## Запуск

### Разработка
```bash
npm run dev
```

### Сборка
```bash
npm run build
```

### Preview
```bash
npm run preview
```
