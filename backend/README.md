# Backend - Delivery App

## Описание

Backend часть приложения доставки.

## Установка

```bash
npm install
```

## Настройка

1. Скопируйте `.env.example` в `.env`
2. Заполните необходимые переменные окружения

## Запуск

### Разработка
```bash
npm run dev
```

### Сборка
```bash
npm run build
```

### Production
```bash
npm start
```
