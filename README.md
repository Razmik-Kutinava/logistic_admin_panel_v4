# Delivery App

Проект приложения для доставки.

## Структура проекта

```
delivery-app/
├── backend/        # Backend приложение
├── frontend/       # Frontend приложение
└── docs/          # Документация
```

## Быстрый старт

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Документация

См. папку `/docs` для подробной документации:
- `ARCHITECTURE.md` - архитектура проекта
- `API.md` - документация API
- `SETUP.md` - инструкции по настройке
